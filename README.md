# Erzsó & István — Wedding

Wedding website for Erzsó and István, hosted on [GitHub Pages](https://pages.github.com/).

**Live site:** https://hovanion.github.io/wedding/

## Customize

| File | What to change |
|------|----------------|
| `index.html` | Names, dates, locations, story text, schedule |
| `styles.css` | Colors, fonts, spacing, hero image |
| `script.js` | Wedding date for the countdown (`WEDDING_DATE`) |

## Local preview

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
